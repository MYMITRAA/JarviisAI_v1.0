## Your Task

Generate a Playwright TypeScript test suite. Follow these STRICT requirements:

### Code Requirements
1. Use `import { test, expect } from '@playwright/test';`
2. Every test must be in a `test.describe` block named after the feature/page
3. Use `page.goto('{{ base_url }}...')` with full URLs
4. Use `await expect(...)` assertions — never bare assertions
5. Add `test.setTimeout(30000)` to every test
6. Use `await page.waitForLoadState('networkidle')` after navigations
7. Include `// JarviisAI: <category>` comment at the top of each test
8. Use stable selectors: prefer `getByRole`, `getByLabel`, `getByPlaceholder`, `getByText` over CSS selectors
9. Every test must be independently runnable (no shared state)
10. Include both happy path AND at least one negative/edge case per feature

### Test Categories to Cover (in priority order)
1. **CRITICAL** — Authentication flows (login, logout, registration) if present
2. **HIGH** — Primary navigation and page loading
3. **HIGH** — Form submissions (valid and invalid data)
4. **MEDIUM** — Interactive elements (buttons, dropdowns, modals)
5. **MEDIUM** — Error states and empty states
6. **LOW** — Responsive behavior hints
